# eatStreet

### The stack

#### Server-side

* **Node.js**
* **Express**
* **MongoDB (with Mongoose)**

#### Client-side

* HTML5, CSS (SCSS) and JS (ES2015)
* **Vue.js**
* Bootstrap 4

#### Development

* Webpack 3
* NPM
* ESLint
* Mocha




### Setup

```
npm i
```

```
// Start development server
npm start
```
